▸ Info

This website explores the crystallographic space-group and point-group datasets generated for httk. The data files used by the site are the current JSON-LD wrapped outputs in data-generators/data, copied into static/data.

Things good to know about space groups

  • There are 230 distinct three-dimensional space-group types. They are commonly identified by their International Tables number, 1-230.
  • Different axis choices, origin choices, and cell choices give different settings of the same space-group type. This explorer can show only the IT reference setting or all tabulated settings.
  • The current static space-group table contains 527 symmetrically unique Hall settings. The companion transformation table is keyed by H-M entry and contains 530 conventional settings from International Tables Volume B table A1.4.2.7.
  • The three extra conventional settings occur in space group 68: three pairs have distinct H-M entries but share the same Hall symbol and the same static symmetry data. The H-M entry field is shown where this distinction matters.
  • Hermann-Mauguin symbols are shown in short, full, and extended forms where available. The modern symbols with the e double-glide notation are preferred, with older forms kept as aliases when present in the source data.
  • Hall symbols are shown using the International Tables convention used by Volume B table A1.4.2.7. Earlier lists can differ, for example by using the lattice symbol S in places where modern tables do not.
